AgentMail 是唯一原生为 Agent 设计的 App，其他都是人类 App 的 Agent 插件。

## 📧 邮件

- **AgentMail** — API-first 邮件，AI Agent 专用（[agentmail.to](https://agentmail.to)）
- **himalaya** — 通用邮箱 CLI，支持 IMAP/SMTP 多账号（[github.com/soy-yo/himalaya](https://github.com/soy-yo/himalaya)）
- **gog** — Gmail / Calendar / Drive CLI（[github.com/chrisnkr/gog](https://github.com/chrisnkr/gog)）

## 📋 任务管理

- **things CLI** — Things 3 命令行，Mac 原生体验（[github.com/AlexZhangJi/things-cli](https://github.com/AlexZhangJi/things-cli)）
- **Todoist API** — 跨平台 Todoist REST API（[developer.todoist.com](https://developer.todoist.com)）

## 📂 笔记与知识库

- **obsidian-cli** — Obsidian 笔记 CLI，本地隐私（[github.com/kbNSClw/obsidian-cli](https://github.com/kbNSClw/obsidian-cli)）

## 📚 文献管理

- **Zotero API** — 学术文献管理器 REST API（[zotero.org/support/dev/web_api](https://www.zotero.org/support/dev/web_api/v3/start)）

## 💬 消息

- **Telegram Bot API** — 开放消息平台，curl 即可调用（[core.telegram.org/bots/api](https://core.telegram.org/bots/api)）
- **Feishu Open API** — 飞书文档 / 知识库 / 消息（[open.feishu.cn](https://open.feishu.cn)）

## 🌐 网络

- **Tavily** — LLM 优化搜索 API，原生为 Agent（[tavily.com](https://tavily.com)）
- **Playwright** — 浏览器自动化，API 覆盖不到时用（[playwright.dev](https://playwright.dev)）
- **markitdown** — PDF/DOCX/PPT 转 Markdown（[github.com/markitdown](https://github.com/markitdown)）

## 🔧 开发

- **gh CLI** — GitHub 官方命令行（[cli.github.com](https://cli.github.com)）
- **Claude Code** — AI 编程助手，通过 ACP Protocol 调用（[claude.ai/code](https://claude.ai/code)）
- **NCBI Datasets CLI** — 基因组 / 基因 / 病毒数据下载（[ncbi.gov/datasets](https://www.ncbi.gov/datasets)）

## 🗄 数据库

- **sqlite3** — 轻量级 .sqlite 文件直接操作
- **pgcli** — PostgreSQL CLI，支持自动补全

## 🧬 环境管理

- **micromamba** — Python/R 环境管理，生信工具链（[mamba.readthedocs.io](https://mamba.readthedocs.io)）
