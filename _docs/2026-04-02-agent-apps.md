---

## 📧 邮件

- **AgentMail** — API-first 邮件平台，AI Agent 专用邮箱基础设施（[agentmail.to](https://agentmail.to)）

## 🌐 网络

- **Tavily** — LLM 优化搜索 API，为 Agent 实时获取网络信息而生（[tavily.com](https://tavily.com)）

## 🔧 开发

- **Claude Code** — AI 编程助手，通过 ACP Protocol 驱动（[claude.ai/code](https://claude.ai/code)）
- **OpenClaw** — AI Agent 框架，支持 Skills、多渠道接入、记忆管理（[openclaw.ai](https://openclaw.ai)）

## 🧬 生信

- **ClawBio** — 首个生信-native AI Agent Skills 库，Local-first（[github.com/ClawBio/ClawBio](https://github.com/ClawBio/ClawBio)）
- **AutoBA** — 全自动多组学分析 Agent，RNA-seq/scRNA-seq/WGS 等（[claw4science.org](https://claw4science.org)）
- **BioDiscoveryAgent** — LLM 驱动，闭环设计遗传扰动实验（[claw4science.org](https://claw4science.org)）

## 📊 数据 & 分析

- **Claw4Science** — 首个系统梳理 AI Agent 科研生态的数据集/平台，覆盖 91 项目、2230 Skills（[claw4science.org](https://claw4science.org)）
